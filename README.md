# C++ Mini Projects Portfolio

This repository contains my beginner C++ projects created while learning programming.

## Projects

### 1. Tic Tac Toe (Player vs AI)
A console-based Tic Tac Toe game where the player competes against a simple AI.

Features:
- Player vs AI gameplay
- Win detection (rows, columns, diagonals)
- Simple AI logic (attack, block, random move)
- Console-based interface

Technologies:
- C++
- Arrays
- Functions
- Random number generation

---

### 2. Task Manager (Console App)
A simple task management application that allows users to manage a list of tasks.

Features:
- Add tasks
- Remove tasks
- Display tasks
- Uses dynamic container (vector)
- Menu-driven interface

Technologies:
- C++
- STL vector
- Functions
- User input handling

---

## Purpose
These projects were created to practice C++ fundamentals such as:
- loops and conditions
- functions
- arrays and vectors
- basic AI logic
- user interaction in console applications
